# Кейс 4. Chunking с overlap

#### Цель

Понять, как сохраняется контекст между соседними чанками.

#### Сценарий

Создаём чанки:

```
Размер чанка: 800
Overlap: 100
```

#### Реализация

```php
<?php

$chunkSize = 800;
$overlap = 100;

$result = [];

for (
    $i = 0;
    $i < count($tokens);
    $i += ($chunkSize - $overlap)
) {

    $result[] = array_slice(
        $tokens,
        $i,
        $chunkSize
    );
}
```

#### Визуализация результата

```
Chunk 1: 1–800

Chunk 2: 701–1500

Chunk 3: 1401–2200
```

#### Что изучаем

* overlap;
* потерю контекста;
* качество поиска.
