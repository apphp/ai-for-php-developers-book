# Кейс 4. Аналитика событий и таймлайнов

#### Сценарий

Есть база событий (новости, факты, изменения).

Пользователь спрашивает:

> «Как развивались санкции против IT-компаний в 2022 году?»

#### Retrieval

* чанки = отдельные события;
* embedding краткого описания;
* фильтр по времени до embedding (важно!).

#### Context building

Контекст упорядочен хронологически, а не по similarity:

```
Ниже приведены события в хронологическом порядке.
Используй их для обобщения, не добавляя новых фактов.

2022-02-24: {{chunk_1}}
2022-03-10: {{chunk_2}}
2022-05-01: {{chunk_3}}
```

#### Почему это важно

Даже если retrieval нашёл события правильно, порядок контекста определяет, будет ли ответ логичным.

#### Ценность кейса

Показывает, что context building – это не «склейка», а бизнес-логика.
