# Кейс 2. Юридический QA: ответы с привязкой к источнику

#### Сценарий

Есть договоры, приложения, дополнительные соглашения.

Юрист или менеджер спрашивает:

> «При каких условиях договор может быть расторгнут в одностороннем порядке?»

#### Почему RAG критичен

* нельзя «догадываться»;
* важна точная формулировка;
* желательно понимать, _из какого пункта_ взят ответ.
* чанки строго по статьям/пунктам;
* метаданные: номер договора, раздел, пункт.

#### Context building

Используется псевдо-JSON, чтобы модель различала текст и структуру:

```
{
  "document": "Contract_2024",
  "section": "Termination",
  "clause": "7.2",
  "text": "{{chunk}}"
}
```

#### Генерация

Модель отвечает в формате:

– краткий вывод

– ссылка на пункт

#### Ценность кейса

Показывает RAG как explainable AI: ответ + источник, а не «мнение модели».
