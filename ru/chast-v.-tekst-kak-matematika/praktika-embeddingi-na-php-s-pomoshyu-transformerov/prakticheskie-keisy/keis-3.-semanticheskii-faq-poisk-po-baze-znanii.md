# Кейс 3. Семантический FAQ / поиск по базе знаний

Сценарий

Есть список вопросов и ответов. Пользователь задаёт вопрос своими словами.

Что демонстрируем

– вопрос ≠ ключевые слова

– поиск по смыслу

– CLS vs mean pooling (можно сравнить)

Как запустить

1. FAQ как массив структур {question, answer}
2. Эмбеддинг считаем только по вопросам
3. Ищем ближайший
4. Возвращаем ответ

Почему это круто

Очень бизнес-реальный кейс, отлично ложится на PHP.
