# Кейс 4. Интеллектуальная навигация по событиям / таймлайнам

Сценарий

Есть события с коротким описанием. Пользователь ищет:

«санкции против IT-компаний»

«религиозные конфликты в регионе»

Что демонстрируем

– эмбеддинги как универсальный индекс

– независимость от языка формулировки

– поиск по коллекции структурированных объектов

Как запустить

1. Массив событий {id, title, description}
2. Эмбеддинг только по description
3. Поиск ближайших
4. Сортировка + вывод

Инженерная ценность

Это уже почти продукт, не учебный пример.
