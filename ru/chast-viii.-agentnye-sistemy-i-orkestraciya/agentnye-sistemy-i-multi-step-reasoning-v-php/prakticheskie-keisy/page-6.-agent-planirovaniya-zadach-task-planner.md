# Page 6. Агент планирования задач (Task Planner)

Этот кейс демонстрирует чистый multi-step reasoning.

#### Сценарий

Пользователь пишет:

```
"Подготовь план изучения машинного обучения на 30 дней."
```

#### Шаги

Агент:

1 разбивает задачу

2 генерирует подзадачи

3 формирует план

#### Пример результата

```
Week 1
Linear regression
Probability basics

Week 2
Neural networks
Backpropagation

Week 3
Transformers
Attention

Week 4
LLM and agents
```
