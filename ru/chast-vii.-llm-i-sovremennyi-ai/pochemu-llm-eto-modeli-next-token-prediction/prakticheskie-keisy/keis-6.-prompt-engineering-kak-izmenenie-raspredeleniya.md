# Кейс 6. Prompt Engineering как изменение распределения

#### Эксперимент

Берём один и тот же вопрос:

```
Explain SQL injection
```

Добавляем контекст:

```
You are a security instructor.
Explain SQL injection in simple terms.
```

Контекст меняет:

P(token \mid context)
