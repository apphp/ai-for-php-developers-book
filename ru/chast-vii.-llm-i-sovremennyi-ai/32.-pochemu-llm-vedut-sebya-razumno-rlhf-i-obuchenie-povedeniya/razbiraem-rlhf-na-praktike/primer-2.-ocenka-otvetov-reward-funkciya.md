# Пример 2. Оценка ответов (reward-функция)

Теперь делаем шаг ближе к RLHF: вместо сравнения двух ответов – даём абсолютную оценку.

#### Пример

```
<?php

function reward(string $answer): float {
    $score = 0.0;

    // длина
    $score += min(strlen($answer) / 100, 2);

    // структурированность
    if (preg_match('/\d+\./', $answer)) {
        $score += 2;
    }

    // наличие пояснений
    if (str_contains($answer, "например")) {
        $score += 1.5;
    }

    // штраф за слишком короткий ответ
    if (strlen($answer) < 30) {
        $score -= 2;
    }

    return $score;
}

$answers = [
    "Используйте PHP.",
    "1. Начните с синтаксиса\n2. Делайте проекты\n3. Практикуйтесь каждый день, например создайте API"
];

foreach ($answers as $a) {
    echo "Score: " . reward($a) . "\n$a\n\n";
}
```

***

#### Инженерный смысл

Вы делаете ключевую вещь:

> превращаете текст → в число

И это ровно то, что делает настоящая reward model.
