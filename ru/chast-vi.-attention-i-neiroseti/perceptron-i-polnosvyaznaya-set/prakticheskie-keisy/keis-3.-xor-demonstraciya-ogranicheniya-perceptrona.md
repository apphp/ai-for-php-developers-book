# Кейс 3. XOR – демонстрация ограничения перцептрона

Это образовательный кейс.

#### Данные

```
$X = [
    [0, 0],
    [0, 1],
    [1, 0],
    [1, 1],
];

$y = [0, 1, 1, 0];
```

Любой линейный перцептрон не сможет это решить.

<br>

Это идеальная точка, чтобы показать:

* геометрию
* почему нужна нелинейность
* зачем скрытые слои
