# Сквозной кейс: распознавание цифр (MNIST)

{% hint style="info" %}
**Сквозной практический кейс**

В этом кейсе в дополнение к отдельным практическим заданиям для глав мы работаем с ещё одной задачей. Эта задача – распознавание рукописных цифр из датасета [MNIST](https://ru.wikipedia.org/wiki/MNIST_\(%D0%B1%D0%B0%D0%B7%D0%B0_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85\)) (Modified National Institute of Standards and Technology).

С каждой новой моделью мы будем возвращаться к этой задаче и улучшать результат.
{% endhint %}

#### **Что мы будем делать в этой главе**

В предыдущей главе мы решали эту задачу при помощи k-ближайших соседей (без обучения). В этой главе мы попробуем её решить при помощи алгоритма деревья решений (Decision Trees).

То есть, вместо того чтобы сравнивать новый объект с уже известными примерами, мы будем строить модель, которая сама формирует правила принятия решений на основе данных. Иначе говоря, если k-NN "смотрит по сторонам" и ищет похожих соседей, то дерево решений задаёт последовательность вопросов, которые шаг за шагом приводят к ответу.
