# Кейс 4. Переуверенная модель как сигнал проблемы

### Кейс 4. Переуверенная модель как сигнал проблемы

#### Сценарий

Модель почти всегда возвращает вероятности около 1.

#### Проверка

```
foreach ($probabilities as $class => $p) {
    if ($p > 0.97) {
        echo "Подозрительно высокая уверенность: $class = $p" . PHP_EOL;
    }
}
```

#### Вывод

Чрезмерная уверенность часто указывает на переобучение или утечку данных.
