# Кейс 5. Обновление уверенности при новых данных

#### Сценарий

Есть базовая вероятность оттока клиента. Приходит новый сигнал.

#### Реализация

```
$prior = 0.3;
$signalStrength = 0.25;

$posterior = min(1.0, $prior + $signalStrength);

echo $posterior;
```

#### Вывод

Вероятность – это живое знание, которое меняется по мере поступления информации.
