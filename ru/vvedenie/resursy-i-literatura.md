# Ресурсы и литература

В этом разделе перечислены полезные инструменты, библиотеки и ссылки для дальнейшего изучения тем, рассматриваемых в этой книге.

#### Официальный сайт книги

* [https://aiwithphp.org](https://aiwithphp.org/)

#### Веб-порталы и учебные пособия

* Javatpoint — бесплатные онлайн-уроки\
  [https://www.javatpoint.com](https://www.javatpoint.com/)
* GeeksforGeeks — портал по информатике для гиков.\
  [https://www.geeksforgeeks.org](https://www.geeksforgeeks.org/)
* W3Schools — онлайн-уроки по веб-разработке\
  [https://www.w3schools.com](https://www.w3schools.com/)

#### Библиотеки и инструменты

* Rubix ML — PHP-библиотека для создания моделей машинного обучения\
  [https://rubixml.org](https://rubixml.org/)
* RubixML/Tensor — PHP-библиотека для создания моделей машинного обучения\
  [https://github.com/RubixML/Tensor](https://github.com/RubixML/Tensor)
* PHP-ML — Легковесная PHP-библиотека для машинного обучения\
  [https://php-ml.readthedocs.io](https://php-ml.readthedocs.io/)
* MathPHP — Мощная современная математическая библиотека для PHP\
  [https://github.com/markrogoyski/math-php](https://github.com/markrogoyski/math-php)
* OpenAI PHP Client — PHP-клиент API, позволяющий взаимодействовать с Open AI API\
  [https://github.com/openai-php/client](https://github.com/openai-php/client)
* LLM Agents PHP SDK — PHP-библиотека, разработанная для упрощения создания и управления агентами на основе языковых моделей (LLM)\
  [https://github.com/llm-agents-php/agents](https://github.com/llm-agents-php/agents)
